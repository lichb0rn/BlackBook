# Exchanges в RabbitMQ

Типы exchange:
- Fanout
	- Направляет копию каждого сообщения во все очереди, связанные  с данной exchange
- Topic
	- Использует pattern matching для маршрутизации
- Direct
- Default Direct Exchange
- Local Random
- JMS Topic
- Consistent Hashing Exchange
- Random exchange
- Recent history exchange
- Headers
