База данных RPM и другая информация об установленных пакетах RPM хранятся в каталоге /var/lib/rpm. 
Вы можете удалить файлы базы данных, начинающиеся с `__db*,` и перестроить их из метаданных, хранящихся в других файлах этого каталога.

Перед началом работы рекомендуется создать резервную копию каталога `/var/lib/rpm`. 
Затем нужно удалить старые файлы `__db*` и перестроить их заново. 

Для этого введите следующие команды:
```sh
cp -r /var/lib/rpm /tmp
cd /var/lib/rpm
rm __db*
rpm --initdb
```
  
Новые файлы ``__db*`` появятся в этом каталоге через несколько секунд.  Выполните простую команду rpm или yum, чтобы убедиться, что базы данных теперь в порядке.
