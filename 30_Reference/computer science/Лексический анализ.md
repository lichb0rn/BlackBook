Сначала код [[Lexer|лексером]] разделяется на токены, которые в дальнейшем будут отданы [[Parser|парсеру]]. 
Допустим есть строка:
```c
let x = 5 + 5;
```

На выходе из лексера может получится что-то вроде этого:
```c
[
	LET,
	IDENTIFIER("x"),
	EQUAL_SIGN,
	INTEGER(5),
	PLUS_SIGN,
	INTEGER(5),
	SEMICOLON
]
```
