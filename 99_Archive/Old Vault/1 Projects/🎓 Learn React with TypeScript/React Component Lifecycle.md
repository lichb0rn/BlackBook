# [[React Component Lifecycle]]


1. **Mount / initial render**
- Компонент первый раз создан
- Начальные значение [[React Component|props]]
2. **Re-render**
- Изменение state
- Изменение props
- re-render родителя
- Изменение [[React Context]]
3. **Unmount**
- Удаление компонента
- Удаление state и props
