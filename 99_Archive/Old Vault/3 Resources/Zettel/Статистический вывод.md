# Статистический вывод

**Статистический вывод** - обобщение информации из [[Выборка|выборки]] для получения представления о свойствах [[Популяция|генеральной совокупности]].

Статистический вывод сосредоточен вокруг предположений вокруг *параметров*.
Например:
- $\mu$ - среднее значение
- $\sigma$ - стандартное отклонение

Выделяют 2 способа обобщения:
- *Оценочный*
- *[[Проверка гипотез]]*
