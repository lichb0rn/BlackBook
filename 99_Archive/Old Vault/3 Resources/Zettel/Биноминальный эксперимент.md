# Биноминальный эксперимент

Биноминальный эксперимент - эксперимент, который имеет следующие признаки:
1. Эксперимент состоит из $n$ одинаковых испытаний.
2. Каждое испытание имеет только 2 взаимоисключающих исхода: успех (success, S), неудача (failure, F).
3. Вероятность успеха в одном испытании равна $p$ и остаётся такой же для всех остальных испытаний. Вероятность неудачи равна $q = 1 - p$.
4. Испытания [[Независимые события|независимы]].
5. Биноминальная [[Случайная величина|случайная величина]] $x$ - число успехов в $n$ испытаниях, где $x = 0, 1,2,...,n$.

## Золотое правило
Если размер [[Выборка|выборки]] большой относительно размера [[Популяция|популяции]], например, при $n/N \geq 0.05$, тогда эксперимент не является биноминальным.
