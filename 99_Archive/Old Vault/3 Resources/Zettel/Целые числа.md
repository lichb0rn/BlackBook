# Теория Чисел

Объединение множества [[Натуральные числа|натуральных чисел]], множества чисел, противоположных натуральным числам, и нуля называется *множеством целых чисел* и обозначается $Z$.

# Дискретная математика

Целые числа могут быть записаны в виде $$a = a_sh^s+a_{s-1}h^{s-1}...+a_0$$
$a$ - целое число
$s$ - количество цифр в числе
$h$ - основание системы счисления

## Пример
$$8925_{10}=8*10^3+9*10^2+2*10^1+5*10^0$$
$$1011_2=1*2^3+0*2^2+1*2^1+1*2^0$$
