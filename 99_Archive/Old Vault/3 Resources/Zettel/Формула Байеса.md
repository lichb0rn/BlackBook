# Теорема Байеса

Пусть событие *А* может наступить при условии появления одного из несовместных событий *$B_1$,$B_2$,...,$B_n$*, образующих [[Полная группа событий|полную группу]]. Поскольку заранее не известно, какое из этих событий наступит, их называют ==гипотезами==.

Допустим, что произведено [[Случайный эксперимент|испытание]], в результате которого появилось событие *А*. Поставим своей задачей определить, как изменились вероятности гипотез. Для этого можно воспользоваться *формулой Байеса*:
$$P_A(B_i)=\frac{P(B_i)P_{B_i}(A)}{P(B_1)P_{B_1}(A)+P(B_2)P_{B_2}(A)+...P(B_n)P_{B_n}(A)}$$
где
$P_{B_i}(A)$ - [[Условная вероятность]] события А при условии B.

*Формулы Байеса позволяют переоценить вероятности гипотез после того, как становится известным результат испытания, в итоге которого появилось событие А.*
