# Логарифм

Если $y = e^x$, то $x$ - показательно степени, в котороую надо возвести $e$, чтобы получить $y$: $$x = log_e y$$

![log1.png](https://ykl-res.azureedge.net/cbe59df6-263b-4617-a284-4a259a5f5cdd/log1.png)

## Базовые операции

- $log_e a + log_e b = log_e ab$
- $nlog_e a=log_e a^n$
