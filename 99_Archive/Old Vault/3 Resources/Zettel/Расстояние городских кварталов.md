Расстояние городских кварталов — метрика, введённая [[Герман Минковский|Германом Минковским]].
Согласно этой метрике, расстояние между двумя точками равно сумме модулей разностей их координат.

Например, для плоскости расстояние между $(p_1, p_2)$ и $(q_1, q_2)$ равно: $|p_1-q_1| + |p_2-q_2|$.

```swift
public func manhattanDistance(from ml: MazeLocation, to goal: MazeLocation) -> Float {
    let xDist = abs(ml.col - goal.col)
    let yDist = abs(ml.row - goal.row)
    return Float(xDist + yDist)
}
```
