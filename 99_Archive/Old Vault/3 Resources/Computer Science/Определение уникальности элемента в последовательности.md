# Наивная реализация
Просто проходим по элементам в двух циклах:
```python
def unique_naive(S):
    """Return True if there are no duplicate elements in sequence S."""
    for j in range(len(S)):
        for k in range(j+1, len(S)):
            if S[j] == S[k]:
                return False
    return True
```
Сложность: $O(n^2)$

## Оптимизация
Добиться улучшения можно, предварительно отсортировав массив.
```python
def unique_optimized(S):
    """Return True if there are no duplicate elements in sequence S."""
    temp = sorted(S)
    for j in range(1, len(temp)):
        if S[j-1] == S[j]:
            return False
    return True
```
Сортировка - $O(nlog\ n)$ + проверка в цикле за $O(n)$. Итого: $O(nlog\ n)$
