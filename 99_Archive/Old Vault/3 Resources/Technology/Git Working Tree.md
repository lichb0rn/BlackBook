# [[Git Working Tree]]

- The working copy or working tree or working directory is the collection of project files on your disk that you work with and modify directly.
- 3 distinct states:
	- `Unmodified`:
		- Haven't change this file since last `commit`
	- `Modified`
		- Changed the file since last `commit`
	- `Staged`
		- Set of changes for the next `commit`
