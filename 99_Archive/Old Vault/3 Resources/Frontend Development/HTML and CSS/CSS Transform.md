# CSS Transform

Свойство `transform` позволяет изменять размер, вращать и перемещать элементы.

### Scale

```css
div:hover {
	transform: scale(1.1);
}
```

Увеличивает размер `div` в 1.1 раз при наведении курсора.

### Skew

```css
#bottom {
	background-color: blue;
	transform: skewX(24deg);
}
```

Наклоняет элемент на 24 градуса вдоль оси X.

### Rotate

```css
div {
	transform: rotate(45deg);
}
```

Поворачивает элемент на 45 градусов.
