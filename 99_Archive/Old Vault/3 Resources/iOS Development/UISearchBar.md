# UISearchbar
`UISearchBar`  предоставляет текстовое поле для текста, кнопку поиска, кнопку закладок и кнопку отмены.
Непосредственный поиск не производит, для этого используется [[Delegate Pattern|делегат]] `UISearchBarDelegate`.
