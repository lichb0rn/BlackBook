# Swift Privacy
В swift определены 5 уровней доступа:
- `internal`: используется по умолчанию. Все объявления видны внутри модуля на глобальном уровне.
- `fileprivate`: объявление видно только внутри файла, в котором оно оределено.
- `private`: объявление видно только внутри области видимости (фигурные скобки), в которой оно определено.
- `public`: объявление видно за пределами модуля, в котором оно определено. Модуль при этом нужно импортировать.
- `open`: если класс объявлен ==open==, то код в другом модуле может наследоваться от него. Объявления ==public== для этого недостаточно.

Для  ==setter== и ==getter== класса можно задать различные уровни доступа. Например, `private(set)` запретит измение переменной, но чтение останется.
