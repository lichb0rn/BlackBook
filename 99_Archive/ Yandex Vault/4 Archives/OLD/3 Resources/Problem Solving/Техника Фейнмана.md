Техника [[Ричард Фейнман|Фейнмана]]:

1. Выбрать тему для изучения
2. Объяснить её как будто для 12-летнего
3. Определить слабые места в объяснении
4. Переделать объяснение так, чтобы устранить слабые места
5. Повторить, пока тему не будет изучена
