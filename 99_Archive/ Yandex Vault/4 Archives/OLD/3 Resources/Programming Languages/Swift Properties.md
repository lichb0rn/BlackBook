# Свойства в языке Swift

Свойства ассоциируют какое-либо значение с [[Swift Class|классом]], [[Swift Structures|структурой]] или [[Swift Enumeration|перечислением]].

- `Stored` свойства хранят занчения констант и переменных
- `Computed` свойства вычисляют значение.

`Computed` свойства могут принадлежать классам, структурам и перечислениям в то вермя, как `Stored` - только классам или структурам.

Также можно определить `observers` для отслеживания изменений в значении свойства и `property wrapper` для переиспользования кода getter'ов и setter'ов.
