**Среднее время:** $O(n^2)$
**Лучшее время:** $O(n)$, если массив уже отсортирован

- Стандартная библиотека [[Swift|Swift]] использует гибридный вариант с сортировкой вставками для коллекций $<20$ элементов.

- Сортировка вставками - один из лучших вариантов, если данные на входе в основном уже отсортированы.

## Код
```swift
public func insertionSort<T>(_ collection: inout T)
where T: BidirectionalCollection & MutableCollection, T.Element: Comparable {
    
    guard collection.count >= 2 else {
        return
    }
    
    for current in collection.indices {
        var shifting = current
        
        while shifting > collection.startIndex {
            let previous = collection.index(before: shifting)
            
            if collection[shifting] < collection[previous] {
                collection.swapAt(shifting, previous)
            } else {
                break
            }
            
            shifting = previous
        }
    }
}
```
