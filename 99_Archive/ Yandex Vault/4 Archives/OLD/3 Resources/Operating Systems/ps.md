`ps` - команда для проверки запущенных процессов.
Полезные опции:
`ps u` , `ps ux`- отображение информации, связанной с активным пользователем.
`ps aux` - все процессы от всех пользователей.

```bash
lichb0rn@sithserver:~$ ps u
USER         PID %CPU %MEM    VSZ   RSS TTY      STAT START   TIME COMMAND
lichb0rn  132090  0.0  0.1   9996  5520 pts/0    Ss   14:22   0:00 -bash
lichb0rn  134896  0.0  0.0  10612  3400 pts/0    R+   14:27   0:00 ps u
```
- `VSZ` -  виртуальный размер процесса, показывает размер процесса изображения (в килобайтах)
- `RSS` - размер страниц памяти, размер программы в памяти.
- `START` - время начала выполнения процесса.
- `TIME` - совокупное используемое системное время.

По умолчанию данные сортируются по PID. Но можно и сменить:
`ps -eo pid,user,group,gid,vsz,rss,comm --sort=-vsz | head`
