```go
map[keyType]valueType
```

Map с ключами типа `string` для хранения целых чисел:
```go
var nilMap map[string]int
```

```go
teams := map[string][]string{
	"Orcas": []string{"Fred", "Sam", "Daphne"},
	"Whales": []string{"Barney", "Wilma", "Pebbles"},
	"Snakes": []string{"Hermione", "Cinderella", "Daisy"},
}
```
