# Формула полной вероятности
==Теорема==. *Вероятность события А, которое может наступить лишь при условии появления одного из несовместных событий $B_1$,$B_2$,...,$B_n$, образующих [[Полная группа событий|полную группу]], равна сумме произведений вероятностей каждого из этих событий на соответствующую условную вероятность события А:*
$$P(A)=P(B_1)P_{B_1}(A)+P(B_2)P_{B_2}P(A)+...+P(B_n)P_{B_n}P(A)$$
