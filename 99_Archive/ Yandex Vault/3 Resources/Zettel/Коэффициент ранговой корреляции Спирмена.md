# Корреляция Спирмена

Предположим, что набор данных содержит точки $(x_1, y_1) ... (x_n,y_n)$. Пусть $rank(x_i)$ является позицией ранга $x_i$ в отсортированной последовательности среди всех $x_i$, таким образом, ранг наименьшего значения - это 1, а наибольшего значения - $n$. Тогда
$$\rho = 1 - \frac{6 \sum d_i^2}{n(n^2-1)},$$
где $d_i = rank(x_i)-rank(y_i)$
