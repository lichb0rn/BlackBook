Возведение двучлена  $a+b$ в степень $n$ можно произвести по формуле бинома Ньютона: $$(a+b)^n = a^n + C_n^1a^{n-1}b+C_n^2a^{n-2}b^2+...+C_n^ka^{n-k}b^k+...+C_n^{n-1}ab^{n-1}+C_nab^n$$, где $C_n^k$ - число [[Сочетания|сочетаний]], которые можно образовать из $n$ элементов по $k$.
