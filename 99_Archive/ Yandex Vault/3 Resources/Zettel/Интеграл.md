# Интеграл

Пусть линия задана уравнением _$y=f(x)$_. Разделим отрезок $ab$ на $n$ равных частей:

![[Pasted image 20210813175053.png]]

- Нужно найти площадь криволинейной трапеции под этой линией:
  _$$F_n=y_0(x_1-a)+y_1(x_2-x_1)+...+y_{n-1}(b-x_{n-1})$$_
  Если ввести обозначения:
  _$$x_1-a = dx_0, x_2-x_1=dx_1, ..., b-x_{n-1}=dx_{n-1}$$_
  тогда формула примет вид:
  _$$F_n=y_0dx_0+y_1dx_1+...y_{n-1}dx_{n-1}$$_
  Искомая площадь есть [[Предел  функции|предел]] суммы при бесконечно большом $n$. Лейбниц ввёл для этого предела обозначение $$\int{y dx}$$
  Фурье усовершенствовал обозначение [[Лейбниц|Лейбница]], придав ему вид
  $$\int\limits_a^b{ydx}$$
  Явно указав начальное и конечное значения $x$.

### Первообразная

Пусть функция $f(x)$ - [[Производная функции|производная]] функции $F(x)$, т.е. $f(x)dx$ - [[Дифференциал|дифференциал]] функции $F(x)$: $$f(x)dx=dF(x)$$

Тогда функция $F(x)$ называется _первообразной_ для функции f(x).
<mark>==Пример==</mark>
$3x^2$ - производая от $x^3$, т.е. $3x^2dx$ - дифференциал функции $x^3$:
$$3x^2dx=d(x^3)$$
В общем виде $x^3+C$, где $C$ - постоянное слагаемое.

## Неопределенный интеграл

Неопределенным интегралом выражения $f(x)dx$ называется наиболее общий вид превообразной функции.
Неопределенный интеграл выражения $f(x)dx$ обозначается $$\int{f(x)dx}$$

- $f(x)dx$ - _подынтегральное выржение_
- $x$ - _переменная интегрирования_
  Слово _неопределенный_ подчеркивает, что в общее выражение первообразной функции входит постоянное слагаемое, которое можно взять произвольно[^1]

[^1]: В противолополжность неопределенному интегралу предел суммы $y_0dx_0+y_1dx_1+...y_{n-1}dx_{n-1}$ называется определенным. Неопределенный интеграл - ==фукнция==, определенный - ==число==.
