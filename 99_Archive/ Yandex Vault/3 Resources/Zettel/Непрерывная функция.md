# Непрерывность функции в точке

Функция $f(x)$ называется _непрерывной_ в точке $x=a$, если соблюдаются следующие два условия:

- при $x = a$ функция $f(x)$ имеет определенное значение $b$;
- при $x \to a$ функция имеет [[Предел  функции|предел]], тоже равный $b$.

При нарушении хотя бы одного из этих условий функция называется _разрывной_ в точке $x=a$.

## Разрыв функции

- Если значение функции $f(x)$ стремится к числу $b_1$ по мере стремления $x$ к $a$ со стороны меньших значений, то число $b_1$ называют _левосторонним пределом функции_. $$\lim_{x \to a - 0}f(x) = b_1$$
- Если значение функции $f(x)$ стремится к числу $b_2$ по мере стремления $x$ к $a$ со стороны больших значений, то число $b_2$ называют _правосторонним пределом функции_. $$\lim_{x \to a + 0}f(x) = b_2$$
  -Величина $|b_2 - b_1|$ называется _скачком_ или _разрывом_.
