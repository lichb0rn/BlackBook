# Предел функции

- Число $b$ называется _пределом_ [[Функция|функции]] $f(x)$ при $x \to a$, если по мере того, как $x$ приближается к $a$ (будь то справа или слева), значение $f(x)$ неограниченно приближается к $b$.
  $$\lim\limits_{x \to a}f(x) = b$$

==Определение==.

- Число $b$ называется _пределом_ функции $f(x)$ при $x \to a$, если абсолютное значение разности $f(x) - b$ остаётся меньшим любого заранее данного положительного числа $\varepsilon$ всякий раз, как абсолютное значение разности $x-a$ при $x$, не равном $a$, меньше некоторого положительного числа $\delta$ (зависящего от$\varepsilon$ ).
