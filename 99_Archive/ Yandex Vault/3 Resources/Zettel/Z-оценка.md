# Z-оценка
- Z-оценка - мера относительного разброса наблюдаемого или измеренного значения, которая показывает, сколько стандартных отклонений составляет его разброс относительного среднего значения. Это безразмерный статистический показатель, используемый для сравнения значений разной размерности или шкалой измерений. 

$$Z = \frac{x_i-\mu}{\sigma},$$
где:
- $x_i$ - случайная величина
- $\mu$ - [[Математическое ожидание дискретной случайной величины|Мат. Ожидание]]
- $\sigma$ - [[Среднее квадратическое отклонение]]

Распределение вероятностей для $z$ называется [[Нормальное распределение#Стандартное нормальное распределение]]
