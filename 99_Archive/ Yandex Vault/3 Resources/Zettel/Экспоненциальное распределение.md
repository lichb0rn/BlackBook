# Экспоненциальное распределение
Экспоненциальное распределение - непрерывное распределение, моделирующее время между двумя последовательными свершениями одного и того же события.
Используется для моделирования времени ожидания или жизненного цикла электронных компонент.

[[Плотность распределения вероятностей|Плотность распределения]]: $$f(x)=\lambda e^{-\lambda x}, x \geq 0, \lambda > 0$$

*$\lambda$* - интенсивность.

[[Математическое ожидание дискретной случайной величины|Мат. ожидание]]: $\mu = \frac{1}{\lambda}$
[[Дисперсия дискретной случайной величины|Дисперсия]]: $\sigma^2=\frac{1}{\lambda^2}$
[[Среднее квадратическое отклонение]]: $\mu = \sigma$
