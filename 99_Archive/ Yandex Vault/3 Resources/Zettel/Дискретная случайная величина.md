# Дискретная случайная величина

*Дискретной* (прерывной) называют [[Случайная величина|случайную величину]], которая принимает отдельные, изолированные возможные значения с определёнными вероятностями. Число возможных значений дискретной случайной величины может быть конечным или бесконечным.

### Случайная величина Бернулли
- Случайная величина, принимающая значения 0 и 1, называется случайной величиной Бернулли.
