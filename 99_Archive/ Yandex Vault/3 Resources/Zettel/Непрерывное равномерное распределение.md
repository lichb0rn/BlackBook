# Равномерное распределение
Непрерывное распределение используется для описания поведения [[Непрерывная случайная величина|непрерывной случайной величины]], чью значения равномерно (однородно) распределены на интервале. Иными словами [[Плотность распределения вероятностей|плотность вероятности]] на этом интервале почти повсюду постоянна.
$$f(x) = \frac{1}{b-a}, a \leq x \leq b$$
![[Pasted image 20210920155635.png]]

[[Математическое ожидание дискретной случайной величины|Мат. ожидание]] : $\mu = \frac{b-a}{2}$
[[Дисперсия дискретной случайной величины|Дисперсия]]: $\sigma^2 = \frac{(b-a)^2}{12}$
