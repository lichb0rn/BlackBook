# Классическая вероятность
*Вероятностью события А* называют отношение числа благоприяствующих этому событию исходов к общему числу всех равновозможных несовместных элементарных исходов, образующих полную группу.
$$P(A)=m/n$$

#### Вероятность появления хотя бы одного события
*Вероятность появления хотя бы одного из событий $A_1$, $A_2$,..., $A_n$, независимых в совокупности, равно разности между единицей и произведением вероятностей противоположных событий $\overline{A_1}$,$\overline{A_2}$,...,$\overline{A_n}$:* $$P(A)=1-q_1q_2...q_n$$
