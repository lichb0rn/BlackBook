# План смены домена

## Подготовительные работы

0. Проверить УЗ администратора СРК  в целевом домене
1. Добавить целевой домен в консоли управления Commvault:
	- Manage -> Securiy -> Identity Servers
2. Добавить УЗ администратора целевого домена в группу master
3. Создать в СРК локальную УЗ с правами администратора (группа master)
4. Задать интервал просто СРК (заранее до начала работ):
	- https://documentation.commvault.com/v11/commcell-console/getting_started_with_blackout_window.html
