Можно добавить в [[Makefile]] строку `include`, чтобы загрузить пременные из `.envrc` для дальнейшего использования:
```sh
include .envrc
MIGRATIONS_PATH = ./migrations
 

## run/api: run the cmd/api application
.PHONY: run/api
run/api:
	go run ./cmd/api -db-dsn=${DB_ADDR}
```

После этого их можно использовать передать в приложение как флаг:
```go
flag.StringVar(&cfg.db.dsn, "db-dsn", "", "PostgreSQL DSN")
```
