# Insight Graph

Insight Graph - система заметок, основанная на [Molecular Notes](https://reasonabledeviations.com/2022/04/18/molecular-notes-part-1/)

## Purpose

Build a system to:

- understand (not just store) knowledge  
- connect ideas  
- generate insights  

## Core Idea

Separate:

- **what exists** → `#type/note`  
- **what you think** → `#type/insight`  

## Note Types

### Source — `#type/source`
Input material (books, articles, courses)

- highlights
- excerpts
- rough notes

### Note — `#type/note`
Things that exist independently:

- concepts (CAP theorem)
- techniques (CQRS)
- methods (Training Max)
- formulas

> One concept = one note

### Insight — `#type/insight`
My thinking:

- explanations
- connections
- realizations

> High-value notes

### Topic — `#type/moc`
Organization layer:

- groups notes
- shows structure


## Example (5/3/1)

- `5/3/1 Forever` → `#type/source`
- `Total Reps Method` → `#type/note`
- `Total reps enable autoregulation` → `#type/insight`
- `Strength Training` → `#type/moc`


## Rules

1. One concept = one `#type/note`  
2. Your thoughts = `#type/insight`  
3. Link notes  
4. Extract, don’t copy  
5. Keep notes small  


## Mental Model

- Sources → input  
- Notes → building blocks  
- Insights → intelligence  
- MOCs → navigation

## Workflow

Source → Note → Insight → MOC
