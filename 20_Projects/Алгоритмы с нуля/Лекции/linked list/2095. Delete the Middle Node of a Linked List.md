## Итерация 1

```go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func deleteMiddle(head *ListNode) *ListNode {
    if head.Next == nil {
        return nil
    }

    slow := head
    // даём фору
    fast := head.Next.Next

    for fast != nil && fast.Next != nil {
        fast = fast.Next.Next
        slow = slow.Next
    }

    // slow стоит за 1 ноду до середины
    slow = slow.Next.Next

    return head
}
```

Тест кейсы не проходят:
```
Input
head = [1,3,4,7,1,2,6]

Output
[1,3,4,7,1,2,6]

Expected
[1,3,4,1,2,6]
```
