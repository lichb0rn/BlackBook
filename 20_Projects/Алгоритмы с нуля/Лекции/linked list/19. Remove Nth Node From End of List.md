## Итерация 1

```go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func removeNthFromEnd(head *ListNode, n int) *ListNode {
    dummy := &ListNode{}
    dummy.Next = head

    first := dummy
    second := dummy

    for i := 0; i < n + 1; i++ {
        first = first.Next
    }

    for first != nil {
        first = first.Next
        second = second.Next
    }

    second.Next = second.Next.Next
    return dummy.Next
}
```

 - Мы вставляем заглушку перед головой списка, чтобы не плодить лишние условия.
 - Идём с двумя указателями:
	 - Первому даём фору на  `n + 1` шагов
	 - Тогда к концу списка, второй остановится за 1 шаг до нужной позиции
