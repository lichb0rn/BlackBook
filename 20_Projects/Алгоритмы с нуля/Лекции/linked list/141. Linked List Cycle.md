## Итерация 1

```go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func hasCycle(head *ListNode) bool {
    // Алгоритм Флойда

    slow := head
    fast := head

    for fast != nil && fast.Next != nil {
        slow = slow.Next
        fast = fast.Next.Next

        if slow == fast {
            return true
        }
    }

    return false
}
```

- Используем [[Алгоритм Флойда]]
- Ставим 2 указателя: быстрый и медленный
	- Быстрый идёт в 2 раза быстрее
	- Если медленный "догнал" быстрый - цикл есть
