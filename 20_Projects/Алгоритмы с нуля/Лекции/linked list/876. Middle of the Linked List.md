- Используем 2 указателя: быстрый (fast), медленный (slow)
- Быстрый - в два раза быстрее медленного
- Когда быстрый дойдёт до конца, медленный будет на середине
	- `fast = current.Next.Next`
	- `slow = current.Next`

## Итерация 1

```go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func middleNode(head *ListNode) *ListNode {
    slow := head
    fast := head

    for fast != nil && fast.Next != nil {
        fast = fast.Next.Next
        slow = slow.Next
    }

    return slow
}
```
