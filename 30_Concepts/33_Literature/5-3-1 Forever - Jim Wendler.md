# Key Ideas
- Тренировки разбиваются на циклы: Leader - Anchor
- Каждая тренировка состоит из основных движений (squat, bench, deadlift, overhead press) + вспомогательные упражнения


## Notes

### Assistance work categories

Вспомогательные упражнения делятся на 3 категории: 
- Push
- Pull 
- Single Leg/Core

Каждая категория обычно состоит из 50-100 повторений.
